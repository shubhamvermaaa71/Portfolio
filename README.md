#Personal_Portfolio

A modern and fully responsive personal portfolio website built using HTML, CSS, and JavaScript featuring smooth animations, interactive UI elements, custom cursor effects, and elegant user experience design.

## Features

- Fully Responsive Design
- Smooth Scrolling Navigation
- Custom Animated Cursor
- Interactive Card Hover Effects
- Scroll Reveal Animations
- Dynamic Hero Section
- Modern Glassmorphism Elements
- Clean and Professional UI
- Mobile Friendly Layout
- Optimized Performance

## Technologies Used

- HTML5
- CSS3
- JavaScript
